# Illustration Folio
A bespoke illustration portfolio featuring:
- Custom-built Lightbox with zoom/pan functionality.
- Responsive gallery with intersection observers for fade-in effects.
- Age-verification modal for restricted content.
本作品集包含限制級 (R-18) 內容。

網頁內建年齡確認機制。

成人內容預設為模糊處理，僅在使用者主動確認已滿 18 歲後才會顯示。

瀏覽時請確保您已符合當地法律規定的成年年齡。
